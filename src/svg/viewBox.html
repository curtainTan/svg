viewBox 原理：

整个世界可以当成一个画板，我们只是通过视口看到画板的一部分

viewBox 的前两个值是用来控制视口的移动

viewBox 的后两个值与 svg 的 width 和 height 值的比来设置画板内容的缩放

案例

viewBox设置为：0 0 100 100，width: 200 height: 200

前两位： 0 0，表示没有偏移

后两位：100 100，对标属性：width 和 height，进行对应的缩放，此时表示内部元素放大 2 倍

viewBox设置为：-10 -10 100 100，width: 200 height: 200

前两位： -10 -10，表示视口向 X 和 Y 轴各移动 -10 个单位(所以你看到的蓝正方形向下偏移了)

viewBox设置为：10 10 100 100，width: 200 height: 200

前两位： 0 0

后两位： 200 200，展示 200 宽高的内容

viewBox设置为：0 0 200 200，width: 100 height: 100

前两位： 0 0

后两位： 200 200，展示 200 宽高的内容
